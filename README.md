# Sales-Insights-Project
The Sales Insights Project focuses on analyzing historical sales data to evaluate product-wise and region-wise performance, identify key sales trends, and generate actionable business insights.
The project involves data cleaning, aggregation, and analysis using SQL, followed by visual exploration and dashboard creation in Excel.

Objectives:  
Analyze overall sales performance across products and regions  
Identify top-performing and underperforming products  
Understand regional sales trends and contribution  
Create a clear and interactive Excel dashboard for decision-making
